📋 Setup Instructions
  1. Go to Google Sheets → create a new blank spreadsheet → copy the ID from the URL
  2. Go to script.google.com → New Project → paste the contents of yoga_apps_script.js
  3. In the script, paste your Spreadsheet ID where it says PASTE_YOUR_SPREADSHEET_ID_HERE
  4. Run the setupSheets() function once to create all tabs
  5. Click Deploy → New Deployment → Web App
    Set: Execute as: Me  |  Who has access: Anyone
  6. Copy the deployment URL and paste it above ↑
  7. Upload this HTML file to Cloudflare Pages (pages.cloudflare.com) — free!
  8. Done! Share the Cloudflare Pages URL as your QR code 🎉
